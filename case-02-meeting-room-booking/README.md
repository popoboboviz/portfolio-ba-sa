# Case 02 — Бронирование переговорки

## Описание
Сервис для сотрудников офиса, который позволяет:
- просматривать доступные переговорные комнаты;
- создавать бронь переговорки на выбранный интервал времени;
- отменять бронь;
- отправлять уведомления участникам встречи (Email/Slack);
- администратору управлять переговорками (добавлять/редактировать/закрывать на ремонт).

## Основные сценарии
1. **Создание брони**
   - пользователь выбирает дату/время и переговорку;
   - система проверяет ограничения (валидность времени, длительность, лимит активных броней);
   - система проверяет конфликты по выбранной переговорке;
   - при успехе создаёт бронь со статусом `BOOKED` и отправляет уведомления.

2. **Отмена брони**
   - пользователь выбирает бронь в списке “Мои брони”;
   - система проверяет правило отмены (например, не менее чем за 10 минут до начала);
   - при успехе обновляет статус на `CANCELLED` и отправляет уведомления.

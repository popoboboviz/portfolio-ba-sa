# Case 01 — Страница брендов одежды (Marketplace)

## Описание
Функциональный блок маркетплейса: отдельная страница категории «Одежда», где пользователь может:
- просматривать бренды списком (логотип + название),
- сортировать бренды по алфавиту (A–Z),
- искать бренд по названию,
- выбирать бренд и переходить на страницу товаров/баннеров бренда.

Также в бизнес-требовании указано, что контент-менеджеры могут создавать/изменять/удалять бренды и привязывать их к коллекции товаров (в этом кейсе проработан пользовательский сценарий страницы).

## Участники и сервисы
- Пользователь
- Front
- Brand-service (список брендов, поиск/сортировка)
- Item-service (товары и баннеры бренда по коллекции)

## Основные сценарии
1) Открытие страницы → загрузка списка брендов → отображение
2) Сортировка A–Z → отображение отсортированного списка
3) Поиск по названию → найдено / ничего не найдено
4) Выбор бренда → загрузка товаров+баннеров → показ страницы бренда / товаров нет
